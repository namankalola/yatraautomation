**Flight Search Automation Framework**

Overview

This project is a Selenium TestNG automation framework developed to automate flight search scenarios across multiple travel booking platforms.

The framework follows the Page Object Model (POM) design pattern and is designed for scalability, maintainability, and easy integration of additional travel websites.

**Features**

Selenium WebDriver Automation

TestNG Test Execution

Page Object Model (POM)

Dynamic Date Selection Utility

Cross-Browser Support

Reusable Element Wrapper Methods

Screenshot Capture on Failure

Data-Driven Testing Support

Extensible Reporting Framework

Configurable Test Data
